**Cafe 좌석 예약 앱**은 편리한 카페 이용의 미래를 위해 만들었어요!
- 카페 좌석의 현황을 볼 수 있어요.
- 카페 좌석을 미리 예약할 수 있어요.
- 다른 사용자의 예약 좌석은 취소할 수 없어요.

## 📚 Tech
- Html, css, javascript, React-Native
- Node.js

## 🌎 Setting
- Visual Code
- Expo 애플리케이션

## 🤔 How To Use
Expo 애플리케이션을 설치 후 QR코드 인식을 통해 앱을 확인할 수 있어요!
- 프로젝트 실행
```
npm install
```

```
npm start
```
- 서버 실행
```
node server.js
```
